# QUANTUM

## 🚀 Extreme 3D/4D Animated Portfolio Website

An award-winning, immersive 3D portfolio website combining cutting-edge 3D graphics, 360-degree animations, and 4D interactive experiences.

### 🎯 Project Vision
Build an Awwwards-style design with advanced animation & interactivity using:
- **Three.js** for 3D rendering
- **React & TypeScript** for UI/state management
- **GSAP** for timeline animations
- **Tailwind CSS** for styling
- **Framer Motion** for smooth interactions

### 🛠️ Tech Stack
- React 18.2+ with TypeScript
- Three.js (r128) + React Three Fiber
- GSAP v3 with ScrollTrigger
- Vite for fast builds
- Tailwind CSS for styling

### 🚀 Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` in your browser.

### 📦 Build

```bash
npm run build
npm run preview
```

